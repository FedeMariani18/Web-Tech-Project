# Web-Tech-Project

TIPOLOGIA DI PROGETTO SCELTO: Attività ricreative (calcetto, cineforum, ecc)

funzionalità:
- ricerca ed iscrizione ad attività ricreative
- aggiunta di una proposta di attività
    - meccanismo dei tag per facilitarne la ricerca
- login admin e user
    - l'admin può bannare utenti o eliminare proposte


opzionali:
- possibilità di visualizzare le proposte sulla mappa
- messaggistica e scambio di amicize / follow
